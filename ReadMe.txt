This Tic Tac Toe game was created by Liam Du Preez(ldu-pree) student at WeThinkCode CapeTown institute.
Thanks to Kaelin Skye Lees(klees) for helping with the circle drawing algorithm.
Instagram: @pocketsizedfox

To create the program run the command "make" in the Tic-Tac-Toe directory.

To run the program just type "./mlc" in the command line and it will open a new windowthen you can start playing.
Once someone wins it will indicate so and then you can press the middle mouse
(mouse wheel) button to restart.
